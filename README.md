# Sereya-Spice-Kitchen
Sereya Spice Kitchen is a modern African fine dining restaurant website designed to showcase the richness of African fusion cuisine in an elegant and contemporary setting. Inspired by East African coastal traditions and bold spice heritage, the platform highlights signature dishes such as Swahili Pilau, Coastal Coconut Fish Curry, and Velvety Cream Soup. The website features a responsive landing page, fixed navigation, a compelling hero section, and an interactive reservation form to ensure a seamless user experience across mobile and desktop devices. Built using HTML5, the project demonstrates clean structure, responsive design principles, and accessible web practices. Sereya Spice Kitchen represents a celebration of African flavors, refined presentation, and warm hospitality in the heart of Nairobi.
# Key features
Responsive landing page (mobile-friendly)
Fixed navigation header
Hero section with value proposition
Signature dish highlights
Reservation form with validation
Accessible navigation (ARIA attributes)
